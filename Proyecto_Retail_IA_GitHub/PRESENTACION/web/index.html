Machine Learning

Requerimiento Especial

¿En qué parte exacta del proyecto aplicamos Clasificación, Regresión y Clustering? Aquí están los 3 puntos clave:

1. Clasificacion (YOLOv8x)
¿Dónde se usa? En las cámaras y la IA (Raspberry Pi). Es el algoritmo encargado de diferenciar un producto de otro basándose en las imágenes. Sabe cómo clasificar visualmente si esa forma verde en el estante es una lata de crema o una bolsa de papas.
2. Regresion (Random Forest / XGBoost)
¿Dónde se usa? En el motor de inventario oculto. Como la visión artificial solo ve la "cara" del producto, usamos modelos de regresión a partir del tamaño y sombra para "adivinar matemáticamente" el número exacto del stock apilado hacia el fondo.
3. Clustering (K-Means)
¿Dónde se usa? En el sistema de alertas de ventas. Después de tener todo el inventario de días, este algoritmo agrupa los productos automáticamente según su nivel de ventas para dar consejos (clusters):
Alta rotacion: Se vende muy rápido, vigilar diario.
Estable: Venta normal, revisar semanalmente.
Baja rotacion: Se mueve lento, hay riesgo de que caduque el producto.
