Vision Computacional para Retail

Sistema de inventario automatico mediante camaras de seguridad PTZ/Domo conectadas a una Raspberry Pi 5 con AI HAT (Hailo-8L). Deteccion de productos por IA con YOLO-World v2.

Objetivo

Detectar, contar e identificar productos en estantes de tiendas retail usando camaras de seguridad CCTV conectadas a una Raspberry Pi 5 con Hailo-8L AI HAT. Las camaras toman fotos, la RPi procesa las imagenes, cuenta productos, actualiza el inventario y borra las imagenes procesadas.

Arquitectura del Sistema

Camaras PTZ / Domo
Raspberry Pi 5 + AI HAT
YOLO-World v2 (Hailo NPU 13 TOPS)
